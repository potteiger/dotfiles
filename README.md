dotfiles
========

![Screenshot](https://github.com/potteiger/dotfiles/raw/trunk/img/screen.png "Screenshot")

 * FreeBSD 13.1-RELEASE
 * `gruvbox` theme applied where possible
 * `sway` wayland compositor
	* with `sway-bar` status bar enabled
 * `alacritty` terminal emulator
 * `zsh` shell
 * `nvim` text editor
	* `lualine` status line (bottom)
	* `bufferline` tabs for buffers along top
	* `scrollbar` for a scrollbar
